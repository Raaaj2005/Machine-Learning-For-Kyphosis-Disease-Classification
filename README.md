# 🦴 Machine Learning For Kyphosis Disease Classification

## 📖 Overview
This project focuses on the application of machine learning in healthcare, specifically building classification models to detect the presence of Kyphosis disease following spinal surgery. The workflow covers the complete data science pipeline, from importing and exploring medical datasets to performing exploratory data analysis, and finally training and comparing Logistic Regression and Random Forest models.

## 🎯 Objective
The primary objective is to develop, train, and evaluate predictive models to accurately classify Kyphosis disease. By preparing the medical data and comparing a baseline Logistic Regression approach against an ensemble Random Forest Classifier, the project aims to identify the most robust algorithmic strategy for this specific medical classification task.

## 🛠️ Tech Stack & Tools
* **Language:** Python
* **Environment:** Jupyter Notebook
* **Machine Learning:** Scikit-Learn (Logistic Regression, Random Forest Classifier)
* **Data Manipulation & Visualization:** pandas, NumPy, Matplotlib, Seaborn

## 🧠 Key Learnings & Skills Demonstrated
* **Problem Formulation:** Understanding the clinical business case and framing the classification objective for Kyphosis detection.
* **Exploratory Data Analysis (EDA):** Importing and investigating the medical dataset to uncover underlying patient patterns.
* **Data Visualization:** Creating insightful plots to understand feature distributions and their relationship with the disease.
* **Data Preparation:** Effectively splitting the dataset into dedicated training and testing sets.
* **Model Training & Comparison:** Implementing and training both a Logistic Regression model and a Random Forest Classifier to evaluate performance differences.
* **Performance Evaluation:** Assessing and comparing the predictive accuracy of the trained classification algorithms.

## 🚀 Getting Started

### 📋 Prerequisites
To run this project, you will need:
* Python 3.x
* Jupyter Notebook or JupyterLab

### ⚙️ Installation & Setup

1. Clone the repository to your local machine:
   git clone https://github.com/Raaaj2005/Machine-Learning-For-Kyphosis-Disease-Classification.git

2. Navigate into the downloaded project folder:
   cd Machine-Learning-For-Kyphosis-Disease-Classification

3. Install the required dependencies:
   pip install pandas numpy scikit-learn matplotlib seaborn jupyter

4. Launch the Jupyter Notebook environment:
   jupyter notebook

5. Open the project's .ipynb file and execute the cells sequentially to visualize the data and train the models.
